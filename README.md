# Dotfiles

## Installation

- `git clone git@github.com:alexvanderbist/dotfiles.git ~/.dotfiles`
- `~/.dotfiles/install.sh`
- `git clone git@github.com:AlexVanderbist/mackup.git ~/.mackup-repo`
- `mackup restore`

⚠️ Manually export Raycast & TablePlus configs
